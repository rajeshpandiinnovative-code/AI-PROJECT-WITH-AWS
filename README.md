# AI Online Academy

A SaaS website platform for schools built with Node.js and Express.

## What this project includes

- Brand: **AI Online Academy**
- Free-tier-ready architecture for launching without cost
- Simple school and student onboarding APIs
- Starter marketing landing page and admin-ready SaaS flow
- Local JSON-based data store for fast prototyping

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server locally:
   ```bash
   npm run dev
   ```
3. Open your browser:
   ```text
   http://localhost:3000
   ```

## SaaS growth path

- Start with a **free tier** for first 100 students
- Add **paid plan** and checkout integration later
- Move to AWS (Elastic Beanstalk, Lambda, or Amplify) when ready to scale

## Notes

- Add your AWS credentials in a `.env` file later if you deploy to AWS
- This starter is built for quick iteration and MVP launch
